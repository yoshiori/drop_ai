Introduction

Engage DropAI, your personal command console for navigating the digital cosmos with Gemini AI. Conceived as a sleek, top-down terminal interface, DropAI materializes on your desktop with a keystroke, ready to channel your queries and commands to the advanced Gemini Pro and Gemini Pro Vision models. It's more than an assistant; it's your direct conduit to AI-powered insights, image analysis, and data synthesis, all wrapped in a customizable, high-performance Electron shell. Features like global visibility toggle, auto-hide, always-on-top, and dynamic theme switching ensure DropAI integrates seamlessly into your digital cockpit, enhancing your workflow with futuristic efficiency.

Installation

Deploying DropAI onto your battle station can be achieved through several methods:

Pre-built Packages (Recommended for most operatives)

Download the latest stable binaries from our Releases Bay:

  • AppImage (Linux): Download the .AppImage file. Grant it execution permission: 
    chmod +x DropAI-*.AppImage
    Then, simply run the file.
  • Debian/Ubuntu (.deb): Download the .deb package and install via: 
    sudo dpkg -i DropAI-*.deb
  • Fedora/RHEL (.rpm): Download the .rpm package and install using: 
    sudo rpm -i DropAI-*.rpm

From Source Code (For developers and space pioneers)

Clone the repository and navigate into the project directory:


git clone https://github.com/yoshiori/drop_ai.git
cd drop_ai

Install dependencies and launch the application:


npm install
npm start

Usage

Once DropAI is initialized, it lurks silently in the background, ready for your command. Summon or dismiss its interface with a global hotkey:


Press F12 to toggle DropAI window visibility.

DropAI is designed for fluid interaction. Its window can be configured to auto-hide when focus is lost, or pinned to remain always on top of your other applications, ensuring constant access to AI assistance. Configure your Gemini API key upon first launch, and you're ready to interface with the AI. Further commands and interactions are intuitive, displayed within its sleek terminal view. Additional keyboard shortcuts for theme switching (Ctrl+T) and other features are available to streamline your operations.
